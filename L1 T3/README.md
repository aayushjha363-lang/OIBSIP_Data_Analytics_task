# Data Cleaning Project – Airbnb NYC Dataset

## Project Overview

This project focuses on **data cleaning**, which involves detecting and correcting errors, missing information, duplicate entries, and inconsistencies within a dataset. The dataset used for this project contains Airbnb listings from New York City and includes issues such as missing values, repeated records, and inconsistent data formats.

Having well-cleaned data is crucial for performing accurate analysis and producing reliable insights. This project demonstrates practical data preparation and cleaning techniques using Python to improve the quality of the dataset.

---

## Objectives

* Recognize the importance of maintaining high-quality data
* Identify and manage missing or incomplete values
* Find and remove duplicate entries in the dataset
* Ensure consistency by standardizing data formats
* Detect and handle unusual or extreme values (outliers)
* Prepare a refined dataset that is ready for further analysis
