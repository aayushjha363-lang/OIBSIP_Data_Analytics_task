# Marketing Analytics: Customer Segmentation

## Project Overview

This project focuses on customer segmentation using marketing analytics and **data cleaning techniques**. The objective is to group customers based on their purchasing behavior and characteristics. The dataset is prepared through proper data cleaning to ensure accuracy and reliability. This analysis helps businesses understand their customers better, improve marketing strategies, and enhance overall customer satisfaction.

---

## Objectives

* Study customer purchasing behavior and patterns
* Clean and prepare the dataset using data cleaning techniques
* Segment customers into meaningful groups after proper data preparation
* Visualize the different customer segments
* Generate actionable insights to support marketing decisions
